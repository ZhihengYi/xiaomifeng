# 小蜜蜂手机销售网（极简手机销售网）
ps：这是以前老师留的JSP作业。

## 运行环境

**1.Eclipse IDE for Enterprise Java and Web Developers 2021**

**2.Tomcat 9.0**

**3.MySQL 8.0以上版本**

可以将mobiledatabase.sql放到D盘根目录下，然后在MySQL程序中输入 source D:/mobiledatabase.sql; 将其导入。
